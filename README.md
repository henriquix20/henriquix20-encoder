## What it does

**Henriquix20 Encoder** processes your video before uploading to TikTok, applying optimizations that reduce server-side compression and preserve the quality and fluidity of your original edit.

Designed for content creators who edit in After Effects, CapCut, Premiere Pro or any professional editor and want their work to look exactly the same after posting.

---

## Features

- **One-click processing** — drag, drop, done
- **Zero re-encoding** — your video is never re-compressed by the app
- **FPS detection** — automatically reads the frame rate from your file
- **Output folder selection** — save anywhere you want
- **Auto-update** — notifies you when a new version is available
- **Lightweight** — single `.exe`, no installation required

## How to use

1. Download and open `Henriquix20 Encoder.exe`
2. Drag your exported video into the app (or click to browse)
3. Choose the output folder (default: same folder as input)
4. Click **⚡ PROCESS VIDEO**
5. Upload the generated `_hx20.mp4` file at [tiktok.com](https://tiktok.com)
6. Before posting: **More options → Upload HD = ON**

## Recommended export settings (After Effects / AME)

For best results, export your video with these settings before processing:

| Setting | Value |
|---|---|
| Format | MP4 |
| Codec | H.264 |
| Profile | High |
| Level | 4.1 |
| Frame Rate | 60fps (constant) |
| Bitrate | VBR, 8 / 12 Mbps |
| Audio | AAC, 192kbps, 48kHz, Stereo |
| Color Space | Rec.709 |
