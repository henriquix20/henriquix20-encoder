## What is this?

**Henriquix20 Encoder** is a tool built for TikTok content creators who want their videos to look exactly the same after posting — no quality loss, no unwanted compression.

It processes your exported video file before upload, applying optimizations that signal TikTok's servers to treat it as a high-quality external upload, reducing the aggressiveness of server-side compression.

---

## Features

- **One-click processing** — drag and drop your video, click process, done
- **Zero re-encoding** — your video is never re-compressed by the app
- **Automatic FPS detection** — reads the frame rate directly from your file
- **Output folder selection** — save the processed file anywhere you want
- **Auto-update notifications** — the app tells you when a new version is available
- **No installation required** — single `.exe` file, just run it

## How to use

**Step 1** — Export your video from After Effects, CapCut, Premiere Pro or any editor

**Step 2** — Open `Henriquix20 Encoder.exe`

**Step 3** — Drag your exported `.mp4` into the app window (or click to browse)

**Step 4** — Choose where to save the output (default: same folder as input)

**Step 5** — Click **⚡ PROCESS VIDEO** and wait for it to finish

**Step 6** — Upload the `_hx20.mp4` file at [tiktok.com](https://www.tiktok.com/upload)

**Step 7** — Before posting, go to **More options → Upload HD = ON**

## Recommended export settings

For the best possible result, export your video with these settings before processing:

| Setting | Recommended value |
|---|---|
| Format | MP4 |
| Codec | H.264 |
| Profile / Level | High 4.1 |
| Frame Rate | Match your composition (24, 30 or 60fps — constant) |
| Bitrate | VBR, Target 8 Mbps / Max 12 Mbps |
| Audio | AAC, 192kbps, 48kHz, Stereo |
| Color Space | Rec.709 (SDR, no HDR) |

> These settings apply to Adobe Media Encoder, but similar options exist in DaVinci Resolve, Final Cut Pro and CapCut Desktop.
